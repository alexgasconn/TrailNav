# 🚀 Guía Rápida - TrailNav en Android

## Instalación Rápida (3 minutos)

### 1️⃣ Abrir en Chrome

```
1. Abre Chrome en tu Android
2. Ve a: https://trailnav.example.com (o localhost si es local)
3. Espera a que cargue completamente
```

### 2️⃣ Instalar como App

```
1. Toca el menú (⋮) arriba a la derecha
2. Selecciona "Instalar aplicación" 
3. Confirma tocando "Instalar"
4. La app aparecerá en tu home screen
```

### 3️⃣ Otorgar Permisos

Cuando te pida:

- ✅ **Ubicación** - Necesario para GPS
- ✅ **Almacenamiento** - Para guardar rutas y mapas
- ⚠️ **Cámara/Micrófono** - Opcional

---

## Uso Básico

### Importar una Ruta GPX

```
1. Toca el botón "IMPORT GPX" en home
2. Selecciona un archivo .gpx de tu teléfono
3. La ruta se analizará automáticamente
4. Aparecerá en "Your Routes" en la pantalla principal
```

**¿Dónde conseguir rutas?**

- AllTrails.com
- Komoot.com  
- Wikiloc.com
- Descarga y busca archivos .gpx

---

### Navegar una Ruta

```
1. En "Your Routes", toca la ruta que quieres hacer
2. Revisa los detalles (distancia, elevation gain, etc)
3. Toca el botón "NAVIGATE" (verde grande)
4. ¡Empieza a navegar!
```

### Durante la Navegación

**Pantalla Principal:**

- 📍 **Distancia restante** - Cuánto falta
- 🏃 **Velocidad actual** - Tu velocidad en tiempo real
- ⏱️ **Tiempo transcurrido** - Cuánto llevas
- ⏳ **ETA** - Tiempo estimado para terminar

**Indicadores:**

- 🟢 **Verde** = Estás en la ruta
- 🔴 **Rojo** = ¡Saliste de la ruta! (vibración doble)
- 🧭 **Brújula** - Tu dirección actual

**Funciones:**

- **Pausa** - Toca play/pausa para pausar
- **Atrás** - Toca X para salir de la navegación
- **Mapa** - Se rota automáticamente según tu dirección

## ✨ Características Especiales para Android

### 🔋 Monitor de Batería

En la pantalla principal verás:

- % de batería
- Si está cargando ⚡ o descargando 🔌

**Consejo:** Activa "Bajo consumo" si batería < 20%

### 📡 Indicador de Conexión

- 📶 Wifi azul = Tiene internet
- ⚠️ Wifi amarillo = Sin internet (pero funciona offline)

### 🧭 Brújula Automática

- El mapa gira automáticamente según dónde mires
- Útil para orientarte rápido

### 📱 Pantalla No se Apaga

Durante la navegación, la pantalla se mantiene siempre encendida:

- ✅ Automático
- ✅ Ahorra batería inteligentemente
- ✅ Se apaga al salir de la navegación

### 📳 Vibración de Alertas

- Una vibración = Volviste a la ruta ✓
- Dos vibraciones = ¡Saliste de la ruta! ✗

## Consejos Prácticos

### Antes de Salir

```
✅ Haz una captura: Status → Internet, GPS, Batería
✅ Descarga los mapas offline (si no lo hiciste)
✅ Carga al 100% tu teléfono
✅ Revisa la ruta en "Análisis" primero
```

### Durante la Ruta

```
✅ Mantén el teléfono en bolsa/mochila para ahorrar batería
✅ Consulta GPS cada 15-30 segundos mínimo
✅ Si vibra dos veces = busca la ruta marcada
✅ Usa auriculares para escuchar alertas
```

### Después

```
✅ Descarga los datos desde "Settings" (opcional)
✅ Deja feedback de la ruta
✅ Comparte tu ruta (datos anónimos)
```

## 🐛 Solucionar Problemas

### "GPS No Funciona"

```
🔍 Diagnóstico:
1. Home screen → Ve el estado GPS
   - Si dice "Ready" ✓ = Funciona
   - Si dice "Unavailable" ✗ = Problema

Soluciones:
1. Activa Ubicación en Android Settings
2. Elige "Modo de precisión alta"
3. Abre el mapa al aire libre (necesita satélites)
4. Espera 1-2 minutos la primer vez
5. Reinicia el teléfono si persiste
```

### "Pantalla Se Apaga"

```
Causas:
- Batería crítica (< 5%)
- Saliste de la app

Soluciones:
1. Carga el teléfono
2. Abre la app nuevamente
3. En Settings desactiva "Timeout de pantalla"
```

### "Usa Demasiada Batería"

```
Consejos para ahorrar:
1. Baja el brillo de pantalla
2. Desactiva Bluetooth/Wifi si no los usas
3. Usa "Bajo consumo"
4. Reduce actualizaciones GPS a cada 30 segundos
5. Descarga mapas offline (más eficiente)
```

### "Los Mapas Se Ven Oscuros"

```
Es así a propósito (modo oscuro para no deslumbrar).

Para cambiar:
1. Ir a Settings → Display
2. Cambiar tema a "Light"
3. Los mapas serán más brillantes
```

### "No Puedo Instalar la App"

```
Requisitos:
✅ Chrome actualizado (versión 90+)
✅ Android 6+
✅ Conexión a internet (para instalar)

Si aún no funciona:
1. Intenta en incognito (Ctrl+Mayús+N)
2. Limpia caché: Settings → Apps → Chrome → Storage → Clear cache
3. Intenta con otro navegador (Edge, Firefox)
```

## 🎯 Mejores Prácticas

### Para Rutas Largas (> 20 km)

```
1. Carga al 100%
2. Lleva powerbank (10,000+ mAh)
3. Baja el brillo a 40-50%
4. Activa "Bajo consumo"
5. Consulta GPS cada 1-2 minutos
```

### Para Rutas Cortas (< 10 km)

```
1. Carga al 80%+
2. Sin necesidad de powerbank
3. Brillo normal (50-70%)
4. Normal consumo de poder
```

### Para Rutas de Noche

```
1. Linterna trasera en el teléfono
2. Pantalla con menos brillo (20-30%)
3. Alarma si te sales > 50m de ruta
4. Descarga mapas previamente
```

## 🎓 Tips Avanzados

### Descargar Mapas Offline

```
1. Ir a "OFFLINE" en navegación inferior
2. Buscar área geográfica
3. Descargar mapa (tarda según tamaño)
4. ✓ Aparece "Downloaded"
5. Úsalo sin internet luego
```

### Compartir Rutas

```
1. En "Recent Routes", mantén pulsada una ruta
2. Toca "Compartir"
3. Elige: Bluetooth, Email, WhatsApp, etc
4. Otra persona puede importarla tocando el archivo
```

### Editar Ruta Localmente

```
1. En "Route Analysis"
2. Toca "Edit Route" (si disponible)
3. Modifica waypoints
4. Guarda los cambios
```
